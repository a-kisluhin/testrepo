Как залить проект на github

  • Создаем репу на github
  • Подключаем репоззиторий к проекту
  • Открываем gitbash в папке проекта
  • Выполняем git remote add origin https://github.com/a-kisluhin/testrepo.git - ccылка на репоззиторий
  • git status - проверка изменений в проекте
  • git add -A - добавляем измененные файлы в память гита
  • git commit -m "COMMIT NAME" - коммитим файлы
  • git push origin master - заливаем файлы в репоззиторий
